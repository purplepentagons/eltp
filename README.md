# Even Less Tree Punching

A progression mod for Minecraft (Fabric, 1.21.1) inspired by a variety of difficulty mods, primarily No Tree Punching.

- Reworks Minecraft progression while keeping modern features intact
- Gives purpose to lesser-used workstations
- Increases tedium and grind of the early to mid game
- Adds new tools and weapons to compensate for the grind
- Various additional configurable mechanics 
- Attempts to decrease the [inventory entropy](https://www.youtube.com/watch?v=GyUL3PbjWcw&t=1756s) of the early game 

Not intended for use in large modpacks. I also do not intend on updating this mod to other versions, as this is just a pet project intended for my own fun.
